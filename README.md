#Exercise 2: What is a program?
#A program is a set of instructions written in a programming language that tells a computer exactly what to do, step by step, to accomplish a task. Like a recipe for the computer.

#Exercise 3: Compiler vs Interpreter
#Compiler: Translates the entire program from source code to machine code all at once before it runs. Example: C, C++. Result is a standalone executable file. Compiler = translating a whole book before reading
#Interpreter: Translates and runs the code _line by line_ as it goes. Example: Python, JavaScript. No separate executable file is made. Interpreter = translating while you read.

#Exercise 8: Human equivalents
#1 Central Processing Unit (CPU) → Your brain. It does all the thinking, calculations, and decision-making.
#2. Main Memory (RAM) → *Your short-term memory*. Holds information you’re actively using right now. Lose it if you get distracted.
#3. Secondary Memory → Your notebook or long-term memory. Stores information permanently that you can recall later.
#4. Input Device → Your eyes, ears, hands. They bring information into the system.
#5. Output Device → Your mouth, hands, facial expressions. They send information out from the system.
